User Directory

A responsive React application that displays a list of users in a clean card-based layout. Users can be viewed, searched, added, edited, and deleted with an intuitive interface.

Features
Display users in a responsive grid layout
Search users by name, email, or company
Add new users
Edit existing user details
Delete users
Loading skeletons for better user experience
Responsive design for desktop and mobile devices
Tech Stack
React.js
JavaScript (ES6+)
CSS3
Vite
Lucide React Icons
